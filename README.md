# Installation

- Drag folder from download into your resources
- Run the provided `new.sql` file on your database
- **IMPORTANT** Setup your config now at this step, including your crypto upgrades. While you can add new hardware to the config down the road, you can not add new categories to the config as the database is reliant on knowing which categories exist on the first load
